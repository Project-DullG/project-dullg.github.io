엔딩 진행 안내

범인은 블랙입니다.

블랙을 지목한 플레이어는 범인 지목 점수를 획득합니다. 블랙은 지목당한 수만큼 은폐 점수를 잃습니다.

비밀 점수 유지 조건

비밀이 다른 플레이어에게 들키거나 지적당했더라도, 본인이 직접 자백하거나 인정하지 않았다면 비밀 점수는 유지됩니다.

조건달성 카드 확인

1. 책장 또는 훈련용 매트: 비밀통로 2. 잠긴 금고: 빈 금고 3. CCTV: 박사 연구실 영상 4. 책상 위: 위치추적 장치
조건달성 카드 요약 ->

BGM 엔딩 페이지에는 배경 음악이 포함되어 있습니다. 하단의 버튼으로 On/Off 할 수 있습니다.

아래 페이지에는 사건의 결말과 진상으로 이어지는 내용이 포함됩니다.

최종 투표 결과에 따른 분기
